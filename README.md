GrowStudy is a student-focused web application designed to help learners plan, track, and improve their daily, weekly, and monthly study routines in a simple and effective way. The platform provides a structured dashboard where users can manage their study plans and stay consistent with their learning goals.

The application includes a secure authentication system (Register & Login), ensuring personalized access for each user. After login, users are redirected to a dashboard that displays different study plans and allows easy navigation across the app.

GrowStudy is built with a clean and responsive UI, making it suitable for both desktop and mobile users. The project focuses on productivity, time management, and user-friendly design, making it especially helpful for students preparing for exams or managing multiple subjects.

Features

User Registration and Login system
Secure session handling
Dashboard with Daily, Weekly, and Monthly study plans
Clean and responsive UI
Simple and beginner-friendly design
Backend integration with database
